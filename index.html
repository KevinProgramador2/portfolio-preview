Sobre Mim

Olá, eu sou o Kevin, Desenvolvedor Web focado em criar aplicações modernas, funcionais e com experiência de usuário eficiente. Desenvolvi um portal de vagas para desenvolvedores utilizando React com Vite, implementando navegação dinâmica com React Router e consumo de API com Axios.

A aplicação conta com sistema de autenticação com diferenciação de perfis (admin e usuário), controle de acesso e funcionalidades completas de CRUD para gerenciamento de vagas, com persistência de dados via LocalStorage e simulação de backend com JSON Server.

Também implementei filtros e busca de vagas, além de páginas dinâmicas, proporcionando uma experiência próxima a plataformas reais do mercado.

Tenho como foco escrever código limpo, modular e escalável, utilizando boas práticas como tipagem com PropTypes e padronização com ESLint.
Atualmente, busco uma oportunidade para atuar com desenvolvimento web, contribuindo com projetos reais e evoluindo continuamente como profissional.
